# **App Name**: PromptRefiner

## Core Features:

- Prompt Input: Accept user-provided prompt via a text input box.
- Prompt Enhancement: Utilize the OpenAI API to rewrite and improve user prompts using the defined system prompt. Apply a temperature between 0.2-0.4.
- Enhanced Prompt Output: Display the enhanced prompt in a dedicated text output box.
- Copy to Clipboard: Allow users to copy the enhanced prompt to their clipboard.

## Style Guidelines:

- Background color: Dark charcoal (#333333) for a modern dark theme.
- Primary color: Vibrant blue (#64B5F6) to provide clarity and structure.
- Accent color: Electric purple (#BA68C8) to signal the transformation.
- Body and headline font: 'Inter' sans-serif for a high contrast, readable presentation.
- Use simple, clear icons for actions like 'Enhance' and 'Copy'.
- Minimalist layout with input box, button, and output box arranged vertically for easy use.
- Subtle fade-in animation on enhanced prompt output.
